title: 选择数据
---
IDE提供了2种选择方法

### 图层面板上选中数据

在图层面板上直接单击想选择的数据，地图上即选中改数据，同时自动高亮1秒。

![图片](./assets/select/select-1.png)

右边编辑区和上方菜单区也会显示选中数据的名字。

![图片](./assets/select/select-3.png)
![图片](./assets/select/select-4.png)

 ### 地图上右键弹出菜单上选中数据

 将鼠标放在在地图想选中的数据上，右键弹出菜单选中。

 ![图片](./assets/select/select-2.png)
